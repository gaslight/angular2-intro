# angular2-jspm-starter

This is yet another starter repo for an angular2 project using jspm and typescript.

## Prerequisites

* Node.js
* jspm

## Usage

  1. Clone the repo

  2. Run
    ```
    npm install
    jspm install
    npm run start
    ```
  3. In separate window start api server
    ```
    npm run api
    ```
  4. Enjoy!
